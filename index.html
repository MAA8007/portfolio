News Classifier

This is an NLP project in which I developed a TensorFlow-based model to classify news headlines into various categories. I implemented an advanced model architecture that included a custom Attention Layer, Bidirectional LSTM, and Dense layers. To ensure effective classification, I managed the one-hot encoding of labels and optimized the model using categorical crossentropy and an increased learning rate. Additionally, I deployed the model with a Gradio interface, enabling real-time, user-friendly headline classification.
